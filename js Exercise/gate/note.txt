Write a function runningBalance(transactions) that takes an array of transaction objects, each with a type ("credit" or "debit") and an amount. Return a new array of numbers showing the balance after each transaction. The starting balance is 0.

const transactions = [
  { type: "credit", amount: 100 },
  { type: "credit", amount: 50  },
  { type: "debit",  amount: 30  },
  { type: "debit",  amount: 90  },
  { type: "credit", amount: 20  },
];

runningBalance(transactions);
// → [100, 150, 120, 30, 50]
